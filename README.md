# 🛒 OLIST E-Commerce Dashboard | End-to-End Data Analytics Project

## 📌 Project Overview

This project is an end-to-end **Data Analytics and Business Intelligence** solution built using the **OLIST Brazilian E-Commerce Dataset**. The project covers the complete analytics pipeline—from **data extraction, cleaning, and transformation using Python**, to **interactive dashboard development in Power BI**.

The dashboard provides actionable insights into sales performance, customer behavior, product categories, payment methods, and delivery performance, helping stakeholders make data-driven business decisions.

---

# 🎯 Project Objectives

* Clean and preprocess raw e-commerce data.
* Perform ETL (Extract, Transform, Load) using Python.
* Build an optimized data model in Power BI.
* Create interactive KPIs and business dashboards.
* Analyze revenue trends and customer behavior.
* Provide business insights through visual analytics.

# 🐍 Python ETL Process

The dataset was first processed using Python before importing it into Power BI.

### ETL Steps

* Loaded multiple CSV files.
* Cleaned missing values.
* Removed duplicate records.
* Converted data types.
* Merged related tables.
* Renamed columns for consistency.
* Created a clean dataset for analysis.
* Exported processed data for Power BI.

# 📈 Dashboard Insights

* Credit Card is the most frequently used payment method.
* Monthly revenue shows seasonal fluctuations.
* Customer review ratings remain consistently high.
* Product categories contribute differently to total revenue.
* Late deliveries account for only a small percentage of orders.
